<head> is the container for all metada information & this data will not be displayed in html page.
  1. <title> : Defines the title of html page in browser. Displays a title for the page in search engine results.
  2. style : Defines internal CSS
  3. link : Defines external CSS
  4. meta : Delement is used to specify which character set is used, page description, keywords, author, and other metadata. Define the character set, description of your web page, keywords for search engines, the author of a page, Refresh document every n seconds. The viewport(HTML5): is the user's visible area of a web page. It varies with the device, and will be smaller on a mobile phone than on a computer screen.
  5. script : Defines client-side JavaScripts.
  6. base : element specifies the base URL and base target for all relative URLs in a page
