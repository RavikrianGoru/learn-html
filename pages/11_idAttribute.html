HTML id attribute

        It specifies a unique id for an HTML element (the value must be unique within the HTML document).
        It can be used in CSS / JavaScript to performa certain taks for a unique element with the specified id value.
        We use "." for class attribute.(Can have spaces. considered to other class attribute)
            ex: class="f1", class="f1 f2"
        We use "#" for id attribute. (Must not have spaces/tabs)
            ex: id="id1"
