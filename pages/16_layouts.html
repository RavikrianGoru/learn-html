<header> defines header of document.
<nav> defines a container for all navigation links.
<section> defines the section in document.
<article> defines independent self-contained article.
<asider> defines content aside from the content (like a sidebar)
<footer> defines footer of document.
<details> defines additional information.
<summary> Defines heading for the <details> tag.

  1. HTML tables(not recommended):
    The purpose of tables to show data in tabular format.
    If we use tabled for layout redesigning is difficult.
