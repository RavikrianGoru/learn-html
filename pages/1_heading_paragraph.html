HTML colors

HTML colors:can be specified using predefined color names, or RGB, HEX, HSL, RGBA, HSLA values.
  • Sample color names: Tomato, Orange, DodgerBlue, MediumSeaGreen, Gray, SlateBlue, Violet, LightGray
  • Sample RGBA value rgb(red, green, blue)(Numeric range 0-255) : rgb(255, 99, 71);
  • Sample HEX value range #000000 to #ffffff: #ff6347;

Cascading Style Sheets(CSS) = style+ colors

We can add css to html in 3 ways. Recommended way is external CSS.
  1. Inline: by using style attribute in html element
  2. Internal: by using <style> element in <head> section.
  3. External: by using external css file.
